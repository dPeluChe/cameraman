🎬 ¿QUÉ ES EL PROYECTO?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Cameraman es una aplicación macOS nativa para grabar y editar pantalla 
con las siguientes características principales:

✅ Grabación de pantalla con ScreenCaptureKit
✅ Grabación simultánea de cámara (overlay)
✅ Audio del sistema y micrófono
✅ Editor de timeline visual
✅ Sistema de export en múltiples formatos (MP4, GIF)
✅ Sistema de zoom automático (keyframes)
✅ App sandboxed con almacenamiento local

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📦 SANDBOX CONFIGURATION:
La app corre con App Sandbox habilitado. Esto afecta:
• Acceso a archivos - Solo container de sandbox + ubicaciones del usuario
• Cámara/mic - Requieren entitlements
• Export - Debe usar NSSavePanel para consentimiento del usuario

Entitlements (CameramanApp.entitlements):
• com.apple.security.app-sandbox
• com.apple.security.device.camera
• com.apple.security.device.audio-input
• com.apple.security.files.user-selected.read-write
• com.apple.security.files.downloads.read-write

📁 FILE PATHS:
• Recordings: ~/Library/Containers/com.dpeluchestudios.CameramanApp/...
• Projects: ~/Library/Containers/.../ProjectStudio/Projects/
• Exports: Temporales en sandbox, luego ubicación del usuario
