Why Cameraman

A recorder that doesn't cost $89
or need a PhD to use.

Screen Studio is polished but paid and closed. OBS is free but has no editor. Cameraman is free, open source, and ships a full timeline editor — plus an MCP server so agents can help.

💰

Free & Open Source

MIT licensed. No subscriptions, no upsells, no telemetry. The whole engine is a Swift package you can reuse.

🔒

Local-First, Zero Cloud

Everything stays on your Mac. No accounts, no uploads. Your recordings and your data never leave your machine.

🤖

Edit With AI Agents

A built-in MCP server exposes 42 tools. Ask Claude or Codex to trim silences, add overlays, change speed — they drive the same editor you do.

🔍

Smart Auto-Zoom

Cursor telemetry detects clicks and pauses, then suggests zoom points automatically. No manual keyframing.

Features

What's built and working.

Not a roadmap. These ship in the current beta.

⏺

Multi-Track Recording

Screen, camera, system audio, and mic captured simultaneously as separate tracks via ScreenCaptureKit.

✂

Multi-Track Timeline

Trim, split, delete, drag-and-drop. Typed tracks for screen, B-roll video, images, audio, and color clips.

⏩

Per-Segment Speed

0.25× to 4× speed per segment. Speed badges visible right on the timeline.

📷

Camera PiP

Circle, rounded rect, capsule masks. Color border. Position per segment. Drag to place anywhere.

🔉

Per-Track Audio

Independent volume (0–300%) and mute per track and per clip. Global track volume sliders in preview and export.

🔍

Auto-Zoom

Cursor telemetry detects clicks and dwell pauses. Zoom suggestions appear automatically. Click to dismiss.

🎨

Clip Effects

Sepia, B&W, brightness, contrast, blur, vignette — targetable to screen, camera, or background. Audio pitch shift too.

📝

Overlays

Arrows, rectangles, lines, text with draw-on and fade-in. Drag in timeline. Full popover inspector for style.

💬

On-Device Transcription

WhisperKit speech-to-text on Apple Silicon. Generates SRT/VTT captions. Model picker from base to large.

🧠

Local Intelligence

Silence detection and chapter suggestions run on-device. One call from an agent and your edit is structured.

📤

Export Presets

Web 1080p H.264, HEVC, 4K 60fps, Portrait, Animated GIF. Live size estimate. Per-segment effects preserved.

💾

Non-Destructive

All edits live in project.json. Source files are never modified. Autosave with undo/redo. Portable bundles.

Edit with AI agents

Your editor,
scripted by an agent.

Cameraman ships a built-in MCP server. Connect Claude Desktop, Claude Code, or Codex and they can inspect projects, split clips, add overlays, transcribe, and export — driving the exact same non-destructive engine as the app.

  • ✓42 tools — projects, clips, tracks, overlays, effects, canvas, export, transcription.
  • ✓Non-destructive by design — agents edit project.json, never your source media.
  • ✓Async export & transcription — poll job status until done.
  • ✓Bundled & signed — the binary lives inside the app. Settings → Integrations auto-detects it.

Get it

Download or build from source.

Grab the latest beta build for macOS, or clone and run in Xcode.

Download for macOS All releases

Build from source

# Clone the repo
git clone https://github.com/dPeluChe/cameraman.git
cd cameraman

# Open in Xcode — scheme "CameramanApp" → Run on "My Mac"
open CameramanApp/CameramanApp.xcodeproj

# Or build the engine standalone
cd EngineKit && swift build

# Build the MCP server standalone
cd MCPServer && swift build -c release

Requires macOS 13 (Ventura) or later · Xcode 15+ · Swift 5.9+. Grant Screen Recording, Camera, and Microphone permissions when prompted. The beta DMG is unsigned — see the README for Gatekeeper bypass steps.
