Why Cameraman

A screen recorder that doesn't cost $89 or require a PhD to use.

💰

Free & Open Source

Screen Studio charges $89. OBS is free but has no editor. Cameraman is both free and includes a full timeline editor.

💻

Local-First, Zero Cloud

Everything stays on your Mac. No accounts, no uploads, no subscriptions. Your recordings, your data.

🔍

Smart Auto-Zoom

Records cursor telemetry and automatically suggests zoom points based on clicks and pauses. No manual keyframing.

✂️

Per-Segment Editing

Split your video and control speed, camera position, audio volume, and overlays independently per segment.

Features

What's actually built and working right now.

⏺

Multi-Track Recording

Screen, camera, system audio, and mic recorded simultaneously as separate tracks.

✂

Timeline Editor

Trim, split, delete segments. Drag media items. Undo/redo with autosave.

⏩

Per-Segment Speed

0.25x to 4x speed per segment. Speed badge visible on timeline.

📷

Camera PiP

Circle, rounded rect, capsule masks. Color border. Position per segment. Drag to place.

🔉

Per-Segment Audio

Independent volume (0-300%) and mute per segment. Global track volume sliders.

🔍

Auto-Zoom

Cursor telemetry detects clicks and pauses. Zoom suggestions appear automatically. Click to dismiss.

🎨

Canvas Effects

Background gradients, blur, solid colors. Video padding, corner radius, shadow.

📝

Overlays

Arrows, rectangles, lines, text. Drag in timeline. Popover inspector for position, scale, rotation, color.

📤

Export Presets

H.264, HEVC, 4K, Portrait, Animated GIF. Per-segment effects included in export.

💾

Non-Destructive

All edits stored in project.json. Source files never modified. Autosave with undo/redo.

Build from source

No binary download yet. Build it yourself in Xcode.

# Clone the repo
git clone https://github.com/dPeluChe/labs-cameraman.git
cd labs-cameraman

# Open in Xcode
open CameramanApp/CameramanApp.xcodeproj

# Select scheme "CameramanApp" → Run on "My Mac"

# Or build EngineKit standalone
cd EngineKit && swift build

Requires macOS 13 (Ventura) or later. Xcode 15+. Grant Screen Recording, Camera, and Microphone permissions when prompted.
