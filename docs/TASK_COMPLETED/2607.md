## Synthetic cursor rendering

- `CursorPlan` (resampled path + click marks) and `CursorPlanGenerator` (normalizes raw telemetry into capture-local space) added in `Sources/EngineKit/Zoom/CursorPlan.swift`.
- `CursorRenderer.swift` added as a `MaskedVideoCompositor` extension: draws a configurable cursor dot and expanding click ripples on top of the composited frame.
- `MaskedVideoCompositionInstruction` extended with `cursorPlan` and `cursorConfig`; `MaskedVideoCompositor` calls `compositeCursor` after overlays.
- Preview pipeline (`PreviewComposition.swift` + `PreviewComposition+StaticClips.swift`) passes `PreviewEngine.cursorPlan` and `Project.syntheticCursor` through every compositor instruction, and enables the compositor when the cursor is enabled.
- Export pipeline (`VideoExportSession+Composition.swift`) gains the same wiring via new `ExportOptions.cursorPlan`; the compositor is forced on when the cursor is enabled.
- Added `CursorPlanTests.swift` and an `ExportOptions` cursor-plan test in `ExportEngineTests.swift`.
